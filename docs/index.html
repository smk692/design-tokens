🎨 Figma Variables 동기화

Figma 파일 키를 설정하면 Variables로 자동 동기화됩니다:

FIGMA_FILE_KEY=<파일키> node sync-to-figma.js

# Figma 파일 URL에서 키 찾기:
# https://www.figma.com/file/XXXXXXXX/파일명
#                              ^^^^^^^^
